Collective Wisdom

Collective Wisdom is an interactive virtual library within the Gottesman Libraries at Teachers College Columbia University.

Created by Adiel Fernandez, and made possible by the Myers Foundation and EdLab, Collective Wisdom is a projection-mapped art installation in the newly launched Smith Learning Theater.

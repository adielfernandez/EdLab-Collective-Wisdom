Collective Wisdom

Collective Wisdom is an installation create by Adiel Fernandez, commissioned by the Edlab at Columbia University's Teachers College.

The physical portion of the installation is an interactive, projection mapped library within the newly developed Learning Theater of the Gottesman Libraries. For more information, visit AdielFernandez.com.
